## 💎 Project Overview
APEX is a purpose-built **Interactive Digideck / Sales Experience Platform**. Unlike static PDF decks or traditional websites, APEX is a specialized sales tool designed for a guided, cinematic journey. 

Built specifically for high-stakes brand partnerships and luxury leasing negotiations, APEX provides a structured environment that establishes scale and destination authority within seconds.

---

## 🎬 Key Features

*   **Slide-Based Navigation**: Fully optimized for full-screen "Slide" views, ensuring focus on one key value proposition at a time.
*   **Precision Snap Scrolling**: High-fidelity section snapping ensures every frame is perfectly optimized for presentation.
*   **Integrated Side Navigation**: An architectural sidebar providing intuitive, non-linear access to any part of the deck.
*   **Keyboard Navigation Support**: Seamlessly navigate through the presentation using arrow keys, designed for professional boardroom environments.
*   **Video-Driven Narrative**: Cinematic background modules that establish the destination's scale and energy before a word is spoken.
*   **Interactive Metrics**: High-impact counters and data points designed to validate business opportunities in real-time.

---

## 🎮 How It Works

APEX is engineered for maximum usability in high-pressure sales environments:

1.  **Universal Controls**:
    *   **Interactive Arrows**: Center-aligned high-visibility controls with pulse animations for clear directional cues.
    *   **Side-Nav Strip**: Quick-jump dots for agile, non-linear storytelling.
    *   **Smart Indicators**: Persistent "PAGE 01 / 08" markers to maintain a professional presentation structure.
2.  **Primary Usage**:
    *   **Live Pitching**: Optimized for shared-screen presentations where the presenter maintains full control of the narrative.
    *   **Interactive Sharing**: A lightweight, fast-loading experience allowing partners to explore the vision at their own pace.
    *   **Presentation Ready**: Designed to be opened and presented instantly during live sales calls with no complex setup.

## 🛠 Tech Stack
Designed for visual fidelity and smooth, hardware-accelerated performance.

*   **Framework**: [React](https://react.dev/) + [Vite](https://vitejs.dev/)
*   **Motion**: [Framer Motion](https://www.framer.com/motion/) (Slide physics and entry animations)
*   **Icons**: [Lucide-React](https://lucide.dev/) (Refined, stroke-based iconography)
*   **UI Components**: Custom Vanilla CSS (Architectural navigation and glassmorphism)

## ✨ Design Philosophy

*   **Zero-Distraction UI**: Removed standard website elements like headers and footers to maximize screen space for cinematic assets.
*   **Narrative Momentum**: Every slide transition and micro-interaction is designed to guide the audience's attention naturally.
*   **Premium Affordance**: Utilizing glassmorphism and subtle gold accents to signal high-value exclusivity.
